Chord Grid

Drum Grid

Four synthesized drum lanes share the same 32-step clock as the chord loop.

Chord Catalog

Define reusable chord symbols as comma-separated lowercase note lists. Names can exist here before they appear in the grid.

Example: chord Cm, notes g3,c4,eb4. Invalid or duplicate catalog rows are ignored until fixed.

Sound Catalog

Pick the sound source for rhythm chops, sustained harmony, and drum pads. WebAudioFont sounds load on first playback and fall back to the internal synth if unavailable.

Drums

Current choices are saved in your local preset. WebAudioFont options require a network connection the first time they load.

Bassline Editor

Edit the bassline with a note pool and a pulse pattern. Pattern x consumes the next note and - rests. Manual _ sustains still work.

The editor uses the current Bass Keyboard octave as its visible range. Change the octave in the sidebar, then reopen or refresh the editor.

Chord/Harmony Editor

Edit Rhythm and Harmony with chord pools and pulse patterns. x consumes the next chord, _ sustains or starts after silence, and - rests.

This editor compiles back into the existing 32-step Rhythm and Harmony grid.
